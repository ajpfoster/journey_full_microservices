## Debug mode

- `npm run debug`

## Devopment mode

- `npm run dev`

## Build project to production

- `npm run build`

## Production mode

- `npm run start`

## Set up the variables general to DOS console

- `SET NODE_ENV=development`
- `SET PORT=3003`

- `SET RABBITMQ_URL=localhost`
- `SET RABBITMQ_PORT=5672`
- `SET RABBITMQ_USERNAME=user`
- `SET RABBITMQ_PASSWORD=1`
